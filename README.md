# Dashboard de Vendas em Excel

## Descrição
Projeto de dashboard de vendas criado no Excel para análise de dados.

## Dados
Incluem:
- Data
- Produto
- Categoria
- Quantidade
- Preço
- Total

## Dashboard
Mostra:
- Vendas por mês
- Vendas por produto
- Vendas por categoria
- Faturamento total

## Como usar
Abra o arquivo Excel e visualize o dashboard.
